# Card Game

## Frontend
- Install: `npm i`
- Dev: `npm run dev`

## Backend
- Install: `npm i`
- Dev: `npm run dev`
- Start: `npm start`

## Root (optional)
- Dev: `npm run dev`

## Deploy
- Ensure SSH key access to the server (no password prompt).
- Run: `chmod +x deploy.sh`
- Deploy: `./deploy.sh`
